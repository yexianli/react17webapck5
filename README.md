# 手动搭建的webpack5 react17项目

## 改项目安装
 npm install

 ## 该项目运行
 npm run start

 ## 该项目已有配置
 1. source-map 开发环境配置生产环境没有配置
 2. 配置了less
 3. 引入了antd 如果不需要用可以卸载通过npm uninstall antd
 4. webpack 5 react 17版本
 5. 配置git不提交的文件在.gitingore文件中
